# 🛡️ Defense of Creed — دفاع عن العقيدة

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **دفاع عن العقيدة والشريعة ضد مطاعن المستشرقين (Defense of Creed and Shariah Against Orientalist Claims)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 defense cards with Quranic references, 8 intellectual responses to common claims, daily habit tracker, knowledge quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 themes (Flame, Night, Sky) with localStorage persistence
- Audiobook narrator with karaoke, voice/speed/pitch, loop, sleep timer
- Responsive design + PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.
